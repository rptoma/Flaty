## MVP  

<br/>

Un MVP presupune un set de caracteristici de baza suficiente pentru a capta atentia potentialilor clienti sau a persoanelor interesate. 


Pe baza validării initiale, nevoile clienților s-au concretizat in : 

- trimitere index
- get index
- adaugare anunt
- get anunturi
- login
- inregistrare
